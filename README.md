# treehouse-fullstack-project-2-pagination-filter

Paginated filter project with exceeds criteria.

- Using progressive enhancement to add pagination
- The page includes a list of 54 students. Programming to display 10 students per page, and add buttons to jump between the lists of 10 students.
- Solution works for any number of student listings, whether there were 5, 25, 55 or 100 student listings.
- Added a search bar to let users search the list of students for a particular student or students.
